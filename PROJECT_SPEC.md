# Техническое задание: Сайт группы "Старый Икарус"

## Обзор проекта

**Название:** Stary Ikarus (Старый Икарус)  
**Тип:** Статический веб-сайт (React 19 + Tailwind CSS 4)  
**Языки:** Русский и Английский (с переключателем)  
**Дизайн-концепция:** Мифологический минимализм

### Визуальные элементы

- **Греческие меандры:** Тонкие линии-разделители между секциями в золоте (#D4AF37)

- **Разделители:** SVG-линии в стиле греческого орнамента
- **Тень:** Мягкие тени для глубины (box-shadow: 0 4px 12px rgba(26, 20, 16, 0.08))

## Структура сайта

### Страницы

1. **Главная (Home)** — Героическая секция с логотипом, краткое описание группы, ссылки на музыку и контакты
2. **О группе (About)** — История группы, философия музыки, биографии участников
3. **Музыка (Music)** — Альбомы, треки, плеер (интеграция с Spotify/SoundCloud)
4. **Медиа (Media)** — Фотогалерея, видео, концертные снимки
5. **Контакты (Contacts)** — Форма обратной связи, социальные сети, информация для букинга

### Навигация

- **Главное меню:** Вверху страницы, горизонтальное, с переключателем языков
- **Переключатель языков:** В верхнем правом углу (RU / EN)
- **Мобильное меню:** Бургер-меню на мобильных устройствах

## Архитектура локализации (i18n)

### Структура файлов локализации

```
client/src/
├── locales/
│   ├── ru.json          # Русский язык
│   └── en.json          # Английский язык
├── contexts/
│   └── LanguageContext.tsx  # Контекст для управления языком
├── hooks/
│   └── useTranslation.ts    # Хук для доступа к переводам
└── ...
```

### Реализация

- **React Context API:** Для управления текущим языком
- **JSON-файлы:** Для хранения переводов (простая и эффективная система)
- **Custom Hook `useTranslation()`:** Для удобного доступа к переводам в компонентах
- **LocalStorage:** Для сохранения выбора языка пользователя

### Пример использования

```tsx
const { t } = useTranslation();

<h1>{t('home.title')}</h1>
<p>{t('home.description')}</p>
```

## Анимации

### Логотип (Главная страница)

- **Движение:** Плавное парящее движение вверх-вниз (3-4 сек цикл)
- **Свечение:** Золотое свечение вокруг логотипа (opacity: 0.3 → 0.7)
- **Технология:** CSS Keyframes + Framer Motion для интерактивных эффектов

### Переходы между страницами

- **Эффект:** Затухание влево, появление справа (300-400ms)
- **Технология:** Framer Motion `AnimatePresence`

### Появление текста при скролле

- **Эффект:** Появление с задержкой (stagger effect)
- **Технология:** Intersection Observer API + Framer Motion

### Линии-разделители

- **Эффект:** Рисуются при загрузке страницы (stroke-dasharray animation)
- **Технология:** SVG stroke-dasharray + CSS animation
